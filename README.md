# Local MP3 Player

A simple and lightweight Flutter application for playing local MP3 files. Users can browse, select, and play audio files stored on their device. The app includes a minimal UI, audio controls, and a responsive design.

## Features

- play local MP3 files with cool ui
- Play, pause, and stop audio
- Seek audio using a slider
- Display audio duration and current position
- Responsive UI
- ios and Android support
